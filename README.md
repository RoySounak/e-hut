# e-hut


### How to Use ⚙️
 > #From your command line, first clone this e-hut repo:

    git clone https://github.com/RoySounak/e-hut/

 > #Go into the repository

    cd e-hut

 > #Then install the dependencies either using NPM or Yarn

    npm install  
    
    yarm install
  
 > #Start development server
  
    npm run dev
    
    yarm dev
