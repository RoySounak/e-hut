# e-hut
#How-To-USE

# Clone this repository
$ git clone https://github.com/RoySounak/e-hut

# Go into the repository
$ cd e-hut

Then you can install the dependencies either using NPM or Yarn:

Using NPM:

# Install dependencies
$ npm install

# Start development server
$ npm run dev
$ npm run watch

Using Yarn:

# Install dependencies
$ yarn

# Start development server
$ yarn dev
